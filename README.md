# 🔐 AI-Based Phishing Detection System

## 📌 Overview
This project is an AI-powered phishing detection system that analyzes email content to identify whether it is **phishing or safe**.  
It combines **Machine Learning (Random Forest)** with **heuristic analysis** such as URL detection and suspicious keyword scanning.

---

## 🚀 Features
- 🤖 Machine Learning (Random Forest Classifier)
- 🔗 URL Detection (identifies suspicious links)
- ⚠️ Keyword Analysis (detects phishing-related words)
- 📊 Confidence Score Output
- 📝 Logging System for analyzed emails
- 🌐 Interactive Web Interface using Flask

---

## 🛠️ Tech Stack
- **Backend:** Python, Flask  
- **Machine Learning:** Scikit-learn  
- **Frontend:** HTML, CSS  
- **Libraries:** Pandas, NumPy
